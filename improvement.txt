What I added/changed

  - src/input/keyboard.c/.h: PS/2 keyboard init, IRQ1 handler, ring buffer. Maps
    printable scancodes; now translates extended 0xE0 arrow scancodes to WASD so
    arrows work.
  - src/kernel.c: Snake command now asks how many players (1–4). Each player
    takes their own run (WASD/arrows). After all runs, the terminal shows each
    score and names the winner(s) by length.
  - src/timer/timer.c/.h: PIT driver, IRQ0 handler, tick counter and busy-wait
    sleep.
  - src/display/screen.c/.h: Simple VGA text helpers (clear, putc, print with
    color).
  - src/idt/idt.c and src/idt/idt.asm: Registered IRQ0 (timer) and IRQ1
    (keyboard) handlers.
  - Makefile: Builds the new modules and adds output dirs.
  - src/kernel.c: Backspace-aware terminal, tiny shell (snake, clear, help,
    quit), and the snake game loop. After exiting the game it reinitializes the
    terminal so the prompt is clean.

  You can rebuild (make) and boot bin/os.bin in QEMU as before; arrows should
  now steer the snake.
